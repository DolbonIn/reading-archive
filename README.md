# Reading Archive · AI Bookshelf

디지털 책장을 둘러보며, PDF에서 자동 생성된 책 요약 프레젠테이션을 감상할 수 있는 공간입니다. PDF 한 권을 올리면 AI가 곧바로 40장 이상의 Tailwind 기반 슬라이드를 만들어 책장에 꽂아 줍니다.

## 책장 둘러보기

- **홈**: <https://dolbonin.github.io/reading-archive/>
  - 마우스를 움직이면 책장이 기울어지고, 책을 선택하면 오른쪽 패널에서 표지와 주요 정보가 나타납니다.
  - 책을 클릭하면 새 탭에서 AI가 만든 프레젠테이션 HTML이 열립니다.
- 책이 아직 없다면 “데이터가 비어 있습니다” 안내가 보입니다. 첫 번째 책을 업로드하면 바로 책장에 등장합니다.

## 프레젠테이션 보기

각 책은 `/presentations/<slug>.html` 경로에 저장됩니다. 책장 UI에서 책을 클릭하거나, 주소창에 직접 입력해 접근할 수 있습니다. 프레젠테이션 페이지는 모바일과 데스크톱 모두에서 동일한 Tailwind 스타일로 렌더링됩니다.

## PDF 업로드 (로컬 사용자)

서버 관리자가 제공하는 로컬 업로드 페이지를 통해 책을 추가할 수 있습니다.

1. 로컬에서 `flask --app local_app.app run --reload` 로 서버 실행
2. 업로드 폼: <http://127.0.0.1:5000/upload>
3. PDF를 끌어다 놓거나 선택하고, 제목/저자/설명 등을 입력해 업로드
4. 업로드가 완료되면 책장 홈(<http://127.0.0.1:5000/>)에 바로 반영됨

업로드 진행 상황은 실시간 로그로 확인할 수 있으며, 완료 시 프레젠테이션 링크도 제공됩니다. (관리자 권한이 없을 경우 직접 업로드는 제한될 수 있습니다.)

## PDF → HTML 프레젠테이션 일괄 변환

- `uploads/` 폴더에 있는 PDF를 한 번에 책장에 올리려면 `python scripts/batch_pdf_to_ppt.py` 를 실행하세요.
- 파일명이 `제목 -- 저자 -- ...` 형식을 따르면 스크립트가 자동으로 제목과 저자를 추출해 전달합니다.
- 실패한 항목은 최대 2번까지 재시도하며, 결과 HTML과 manifest 갱신은 기존 파이프라인(`run_pipeline`)을 그대로 사용합니다.
- WSL에서 `uploads/` 디렉터리 전체를 읽지 못할 때는 스크립트가 자동으로 Windows 경로를
  호출해 `.auto_file_list.txt`를 생성하고 그 목록을 사용합니다.
- 자동 생성이 안 될 경우(예: Windows 명령 실행이 차단된 환경)에는 직접 `uploads/filelist.txt`
  를 만든 뒤 `--file-list uploads/filelist.txt` 옵션과 함께 실행하면 됩니다.
